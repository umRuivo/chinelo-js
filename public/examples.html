Examples: Controller Creation

This section demonstrates how to create a new controller in Chinelo.js.

Step 1: Create the Controller File

Create a new JavaScript file in the src/controllers/ directory. For example, let's create src/controllers/example.js:

// src/controllers/example.js

export function hello(req, res) {
    res.send('Hello from Example Controller!');
}

export function greet(req, res) {
    const name = req.query.name || 'Guest';
    res.send(`Greetings, ${name}!`);
}

greet.httpMethod = 'GET';
greet.routeParams = ['name'];

Step 2: Access the Routes

Once the server is running, Chinelo.js will automatically create routes based on your controller file and function names:

  • To access the hello function: /example/hello
  • To access the greet function: /example/greet/:name (e.g., /example/greet/John) or /example/greet?name=Jane

Remember to restart your server if you are not using hot-reloading for changes to take effect.
